# Dashboards en VizProo
Una de las características destacadas de VizProo es la capacidad de crear dashboards interactivos que combinan múltiples gráficos en una sola vista. Esto permite a los usuarios analizar y comparar diferentes conjuntos de datos de manera eficiente.

Para ello visita el notebook [dashboards](../../examples/Dashboard.ipynb) ubicado en la carpeta "examples" del repositorio de VizProo. Este notebook contiene ejemplos prácticos que te guiarán a través del proceso de creación y uso de dashboards en VizProo.
