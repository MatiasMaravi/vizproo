# VizProo
¡Bienvenido a la documentación oficial de VizProo!
VizProo es una biblioteca de visualización de datos en Python diseñada para facilitar la creación de gráficos interactivos y atractivos. Con VizProo, puedes transformar tus datos en visualizaciones impactantes con facilidad.

Vizpro está basado en el uso de ipywidgets con D3.js para ofrecer una experiencia interactiva dentro de entornos Jupyter.

## ¿Por qué usar VizProo?
- **Interactividad**: Muchas librerias de visualización ofrecen gráficos estáticos. VizProo permite a los usuarios interactuar con los gráficos, explorar datos y obtener información de manera dinámica.
- **Fácil de usar**: Con una sintaxis simple y clara, VizProo facilita la creación de gráficos complejos sin necesidad de conocimientos avanzados en visualización de datos.
- **Integración con Jupyter**: VizProo está diseñado para funcionar perfectamente en entornos Jupyter Notebook y JupyterLab, lo que lo convierte en una herramienta ideal para científicos de datos y analistas que trabajan en estos entornos.
- **Creación de Dashboards**: VizProo permite combinar múltiples gráficos interactivos entre ellos en un solo dashboard, facilitando la presentación y análisis de datos complejos.
